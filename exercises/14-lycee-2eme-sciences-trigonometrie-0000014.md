**Démontrer les identités suivantes**

**1)** $$\sin^6 \alpha + \cos^6 \alpha = 1 - \dfrac{3}{4} \sin^2 2\alpha$$

**2)** $$\dfrac{1 + \sin 2\alpha + \cos 2\alpha}{1 + \sin 2\alpha - \cos 2\alpha} = \cot \alpha$$

**3)** $$(\sin \alpha + \sin \beta)^2 + (\cos \alpha + \cos \beta)^2 = 4 \cos^2 \dfrac{\alpha - \beta}{2}$$

**4)** $$\tan \alpha + \tan 2\alpha - \tan 3\alpha = - \tan \alpha \tan 2\alpha \tan 3\alpha$$

**5)** $$\dfrac{2 \sin \alpha - \sin 2\alpha}{2 \sin \alpha + \sin 2\alpha} = \tan^2 \dfrac{\alpha}{2}$$

**6)** $$\dfrac{\sin \alpha + 2 \sin 3\alpha + \sin 5\alpha}{\sin 3\alpha + 2 \sin 5\alpha + \sin 7\alpha} = \dfrac{\sin 3\alpha}{\sin 5\alpha}$$

**7)** $$\sin^2 3\alpha - \sin^2 2\alpha = \sin 5\alpha \sin \alpha$$

**8)** $$\dfrac{\sin \alpha - \sin 3\alpha - \sin 5\alpha + \sin 7\alpha}{\cos \alpha - \cos 3\alpha + \cos 5\alpha - \cos 7\alpha} = - \tan 2\alpha$$

**9)** $$\dfrac{1}{\tan 3\alpha - \tan \alpha} - \dfrac{1}{\cot 3\alpha - \cot \alpha} = \cot 2\alpha$$

**10)** $$\sin \alpha + \sin \beta + \sin \gamma - \sin(\alpha + \beta + \gamma) = 4 \sin \dfrac{\alpha + \beta}{2} \sin \dfrac{\beta + \gamma}{2} \sin \dfrac{\alpha + \gamma}{2}$$

**11)** $$\sin \alpha + \sin 3\alpha + \sin 5\alpha + \sin 7\alpha = 4 \cos \alpha \cos 2\alpha \sin 4\alpha$$

**12)** $$\dfrac{\sin 3\alpha \cos^3 \alpha + \cos 3\alpha \sin^3 \alpha}{3} = \dfrac{\sin 4\alpha}{4}$$

**13)** $$\dfrac{\sin 2\alpha - \sin 3\alpha + \sin 4\alpha}{\cos 2\alpha - \cos 3\alpha + \cos 4\alpha} = \tan 3\alpha$$

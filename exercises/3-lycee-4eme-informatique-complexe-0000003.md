Mettre sous forme algébrique les nombres complexes suivants et calculer leurs modules:

$$z_1 = (2+5i)+(i+3) \quad z_2 = 4(-2+3i)+3(-5-8i) \quad z_3 = (2-i)(3+8i)$$

$$z_4 = (1-i)\overline{(1+i)} \quad z_5 = i(1-3i)^2 \quad z_6 = (1+i)^3$$

$$z_7 = \dfrac{1}{1+i} \quad z_8 = \dfrac{1-2i}{3+i} \quad z_9 = \dfrac{(3+5i)^2}{1-2i} \quad z_{10} = \left(\dfrac{1+i}{2-i}\right)^2+\dfrac{3+6i}{3-4i}$$

---

## Solution


L'objectif est de mettre les nombres complexes suivants sous la forme $z = a + ib$ et de calculer leur module $|z| = \sqrt{a^2 + b^2}$.



1.  **Pour $z_1 = (2+5i)+(i+3)$**

    **Forme Algébrique :**
    On regroupe les parties réelles et imaginaires :
    $$z_1 = (2+3) + (5+1)i = 5 + 6i$$

    **Module :**
    $$|z_1| = \sqrt{5^2 + 6^2} = \sqrt{25 + 36} = \sqrt{61}$$



2.  **Pour $z_2 = 4(-2+3i)+3(-5-8i)$**

    **Forme Algébrique :**
    $$z_2 = (-8 + 12i) + (-15 - 24i)$$
    $$\Longrightarrow z_2 = (-8 - 15) + (12 - 24)i = -23 - 12i$$

    **Module :**
    $$|z_2| = \sqrt{(-23)^2 + (-12)^2} = \sqrt{529 + 144} = \sqrt{673}$$


3.  **Pour $z_3 = (2-i)(3+8i)$**

    **Forme Algébrique :**
    $$z_3 = 6 + 16i - 3i - 8i^2$$
    
    Comme $i^2 = -1$, alors

    $$z_3 = 6 + 13i + 8 = 14 + 13i$$

    **Module :**
    $$|z_3| = \sqrt{14^2 + 13^2} = \sqrt{196 + 169} = \sqrt{365}$$



4.  **Pour $z_4 = (1-i)\overline{(1+i)}$**

    **Forme Algébrique :**
    Le conjugué de $(1+i)$ est $(1-i)$, donc: 

    $$z_4 = (1-i)(1-i) = (1-i)^2$$
    $$ = 1 - 2i + i^2 = 1 - 2i - 1 = -2i$$

    **Module :**
    $$|z_4| = |-2i| = 2$$


5.  **Pour $z_5 = i(1-3i)^2$**

    **Forme Algébrique :**
    On calcule d'abord le carré $(1-3i)^2 = 1 - 6i + 9i^2 = 1 - 6i - 9 = -8 - 6i$.

    $$z_5 = i(-8 - 6i) = -8i - 6i^2 = 6 - 8i$$

    **Module :**
    $$|z_5| = \sqrt{6^2 + (-8)^2} = \sqrt{36 + 64} = \sqrt{100} = 10$$


6.  **Pour $z_6 = (1+i)^3$**

    **Forme Algébrique :**
    $$z_6 = 1^3 + 3(1)^2(i) + 3(1)(i)^2 + i^3$$
    $$z_6 = 1 + 3i - 3 - i = -2 + 2i$$

    **Module :**
    $$|z_6| = \sqrt{(-2)^2 + 2^2} = \sqrt{4+4} = \sqrt{8} = 2\sqrt{2}$$



7.  **Pour $z_7 = \dfrac{1}{1+i}$**

    **Forme Algébrique :**
    On multiplie par le conjugué $(1-i)$ :

    $$z_7 = \dfrac{1-i}{(1+i)(1-i)} = \dfrac{1-i}{1^2+1^2} = \dfrac{1}{2} - \dfrac{1}{2}i$$

    **Module :**
    $$|z_7| = \sqrt{\frac{1}{4} + \frac{1}{4}} = \dfrac{\sqrt{2}}{2}$$


8.  **Pour $z_8 = \dfrac{1-2i}{3+i}$**

    **Forme Algébrique :**
    On multiplie par le conjugué $(3-i)$ :

    $$z_8 = \dfrac{(1-2i)(3-i)}{(3+i)(3-i)} = \dfrac{3 - i - 6i + 2i^2}{10} = \dfrac{1 - 7i}{10} = \dfrac{1}{10} - \dfrac{7}{10}i$$

    **Module :**
    $$|z_8| = \dfrac{|1-2i|}{|3+i|} = \dfrac{\sqrt{5}}{\sqrt{10}} = \dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}$$


9.  **Pour $z_9 = \dfrac{(3+5i)^2}{1-2i}$**

    **Forme Algébrique :**
    Numérateur : $(3+5i)^2 = -16 + 30i$.

    $$z_9 = \dfrac{(-16+30i)(1+2i)}{1^2+2^2} = \dfrac{-16 -32i +30i -60}{5} = \dfrac{-76 -2i}{5}$$
    $$z_9 = -\dfrac{76}{5} - \dfrac{2}{5}i$$

    **Module :**
    $$|z_9| = \dfrac{|3+5i|^2}{|1-2i|} = \dfrac{34}{\sqrt{5}} = \dfrac{34\sqrt{5}}{5}$$


10.  **Pour $z_{10} = \left(\dfrac{1+i}{2-i}\right)^2+\dfrac{3+6i}{3-4i}$**

        Terme 1 : $\dfrac{1+i}{2-i} = \dfrac{(1+i)(2+i)}{5} = \dfrac{1+3i}{5}$.
    
        Au carré : $\left(\dfrac{1+3i}{5}\right)^2 = \dfrac{-8+6i}{25}$.

        Terme 2 : $\dfrac{3+6i}{3-4i} = \dfrac{(3+6i)(3+4i)}{25} = \dfrac{9+12i+18i-24}{25} = \dfrac{-15+30i}{25}$.

        **Forme Algébrique :**
        $$z_{10} = \dfrac{-8+6i}{25} + \dfrac{-15+30i}{25} = \dfrac{-23+36i}{25}$$
        $$z_{10} = -\dfrac{23}{25} + \dfrac{36}{25}i$$

        **Module :**
        $$|z_{10}| = \dfrac{1}{25}\sqrt{(-23)^2 + 36^2} = \dfrac{\sqrt{1825}}{25} = \dfrac{5\sqrt{73}}{25} = \dfrac{\sqrt{73}}{5}$$

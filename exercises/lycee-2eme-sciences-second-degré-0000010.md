**Résoudre dans $\mathbb{R}$ les équations suivantes. Les équations avec un * sont particulièrement difficiles.**

**1)** $$\dfrac{21}{x^2 - 4x + 10} - x^2 + 4x = 6$$

**2)** $$\dfrac{4}{x^2 + 4} + \dfrac{5}{x^2 + 5} = 2$$

**3)** $$(x^2 - 6x)^2 - 2(x - 3)^2 = 81$$

**4)** $$\dfrac{24}{x^2 + 2x - 8} - \dfrac{15}{x^2 + 2x - 3} = 2$$

**5)** $$7\left(x + \dfrac{1}{x}\right) - 2\left(x^2 + \dfrac{1}{x^2}\right) = 9$$

**6)** $$\dfrac{x^2 + 2x + 1}{x^2 + 2x + 2} + \dfrac{x^2 + 2x + 2}{x^2 + 2x + 3} = \dfrac{7}{6}$$

**7)** $$20\left(\dfrac{x-2}{x+1}\right)^2 - 5\left(\dfrac{x+2}{x-1}\right)^2 + 48\dfrac{x^2-4}{x^2-1} = 0$$

**8)** $$(x^2 + 2x)^2 - (x+1)^2 = 55$$

**9)** $$(x^2 + x + 1)(x^2 + x + 2) - 12 = 0$$

**10)*** $$(x^2 - 5x + 7)^2 - (x-2)(x-3) = 0$$

**11)*** $$(x-2)(x+1)(x+4)(x+7) = 19$$

**12)*** $$(2x^2 + 3x - 2)(5 - 6x - 4x^2) = -5(2x^2 + 3x + 2)$$

**13)** $$x^4 - 13x^2 + 36 = 0$$

**14)** $$2x^8 + x^4 - 15 = 0$$

**15)** $$(2x - 1)^6 + 3(2x - 1)^3 = 10$$

**16)*** $$(1 + x)^8 + (1 + x^2)^4 = 2x^4$$

**17)** $$(x - 2)^6 - 19(x - 2)^3 = 216$$

**18)** $$\dfrac{12x + 1}{6x - 2} - \dfrac{9x - 5}{3x + 1} = \dfrac{108x - 36x^2 - 9}{4(9x^2 - 1)}$$

**19)** $$\dfrac{1}{2x + 3} - \dfrac{1}{x^2 - 16} + \dfrac{1}{2x^2 + 11x + 12} - \dfrac{x - 8}{2x^3 + 3x^2 - 32x - 48} = 0$$

**20)** $$\dfrac{x^2 + x + 1}{x^2 - x + 1} = \dfrac{7}{9} \dfrac{x + 1}{x - 1}$$

**21)** $$\dfrac{x + 1}{2(x - 1)} = \dfrac{9}{2(x + 4)} + \dfrac{1}{x - 1}$$

**22)** $$\dfrac{x^2}{x^2 - 4} + \dfrac{x + 1}{2(x - 2)} = \dfrac{1}{2 - x} - \dfrac{1}{x + 2}$$

**23)** $$\dfrac{4x^2 + 29x + 45 - (x + 1)(2x + 15)}{(2(x - 1))^2 - 2(x + 1)(x - 2)} = \dfrac{(x + 1)(x + 5)}{(x - 1)(x - 2)}$$

**24)*** $$\dfrac{(a - x)^4 + (x - b)^4}{(a + b - 2x)^2} = \dfrac{a^4 + b^4}{(a + b)^2}$$

**25)** $$\dfrac{1}{x^2 - 2x + 2} + \dfrac{2}{x^2 - 2x + 3} = \dfrac{6}{x^2 - 2x + 4}$$

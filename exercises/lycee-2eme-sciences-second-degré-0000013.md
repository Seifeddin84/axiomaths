**Résoudre dans $\mathbb{R}$ les équations suivantes.**

**1)** $$\sqrt{x+1} = 8 - \sqrt{3x+1}$$

**2)** $$\sqrt{x + \sqrt{x+11}} + \sqrt{x - \sqrt{x+11}} = 4$$

**3)** $$\sqrt{17+x} - \sqrt{17-x} = 2$$

**4)** $$\sqrt{3x+7} - \sqrt{x+1} = 2$$

**5)** $$\sqrt{25-x} = 2 - \sqrt{9+x}$$

**6)** $$\sqrt{x^2+1} + \sqrt{x^2-2x+3} = 3$$

**7)** $$\sqrt{x^2+x-5} + \sqrt{x^2+8x-4} = 5$$

**8)** $$\sqrt{x^2+x+1} = \sqrt{x^2-x+1} + 1$$

**9)** $$(x^2-4) \sqrt{x+1} = 0$$

**10)** $$\sqrt{4x-3} + \sqrt{5x+1} = \sqrt{15x+4}$$

**11)** $$\sqrt{x+5} + \sqrt{x+3} = \sqrt{2x+7}$$

**12)** $$\sqrt{4-x} + \sqrt{5+x} = 3$$

**13)** $$\sqrt{4x+2} + \sqrt{4x-2} = 4$$

**14)** $$\sqrt{x-\sqrt{x-2}} + \sqrt{x+\sqrt{x-2}} = 2$$

**15)** $$\sqrt{x+7} - x + 3 = 0$$

**16)** $$\sqrt{x^2 + 9} - \sqrt{x^2 - 7} = 2$$

**17)** $$\sqrt{3x^2 + 5x + 8} - \sqrt{3x^2 + 5x + 1} = 1$$

**18)** $$\sqrt{3x^2 - 2x + 15} + \sqrt{3x^2 - 2x + 8} = 7$$

**19)** $$\sqrt{Ax^2 + Bx + C} + \sqrt{Ax^2 + Bx + C_1} = p$$

**20)** $$\dfrac{\sqrt{21+x} + \sqrt{21-x}}{\sqrt{21+x} - \sqrt{21-x}} = \dfrac{21}{x}, \quad x \neq 0$$

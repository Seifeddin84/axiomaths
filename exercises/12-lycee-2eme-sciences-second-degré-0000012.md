**Résoudre dans $\mathbb{R}$ les équations suivantes.**

**1)** $$|x| = x + 2$$

**2)** $$|-x + 2| = 2x + 1$$

**3)** $$|x - 1| + |x - 2| = 1$$

**4)** $$|x - 1| + |x + 2| - |x - 3| = 4$$

**5)** $$|x - 1| - 2|x + 2| + 3|x - 3| = 4$$

**6)** $$\left| 2 - \left| 1 - |x| \right| \right| = 1$$

**7)** $$\left| \dfrac{x+1}{x-1} \right| = 1$$

**8)** $$|5x - x^2 - 6| = x^2 - 5x + 6$$

**9)** $$|x^2 - 1| = -|x| + 1$$

**10)** $$\left| \dfrac{1}{2}x^2 - 2x + \dfrac{3}{2} \right| + \left| \dfrac{1}{2}x^2 - 3x + 4 \right| = \dfrac{3}{4}$$
